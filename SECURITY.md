# Security Policy

## Supported Versions

Report vulnerabilities against the latest `main` / default branch.

## Reporting a Vulnerability

Do **not** open a public issue for security bugs.

Email the maintainers via GitHub Security Advisories on this repository
(Preferred: **Security → Report a vulnerability** on GitHub).

Include: affected component, reproduction steps, impact assessment.

You should receive an acknowledgment within 7 days.
